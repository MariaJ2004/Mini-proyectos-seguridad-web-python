# Mini-proyectos-seguridad-web-python
 Este repositorio contiene tres mini proyectos de Python, ideales para aprender conceptos de **HTTP, seguridad web y crawling**.   Cada proyecto es independiente y tiene su propia carpeta con código y README. A futuro se planea agregar más mini proyectos relacionados, ampliando las herramientas y funcionalidades disponibles.
 ## Proyectos incluidos

1. **Analizador Headers**  
   Herramienta que realiza peticiones HTTP y muestra los códigos de estado y headers de la respuesta de cualquier URL.

2. **Verificador URLs**  
   Herramienta que verifica la disponibilidad de varias URLs mostrando su código de estado HTTP y manejando errores comunes.

3. **Mini Crawler**  
   Mini crawler que visita una URL y extrae todos los enlaces (`<a href="...">`) presentes en la página, evitando duplicados.
   
## Aprendizajes
- Manejo de requests HTTP y códigos de estado
- Análisis de headers de seguridad
- Parsing básico de HTML y extracción de links
